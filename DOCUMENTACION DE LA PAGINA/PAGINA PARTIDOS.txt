🔄 CÓMO ACTIVAR/DESACTIVAR PARTIDOS
✅ Para MOSTRAR un partido jugado:

Elimina SOLO los caracteres /*del inicio
Elimina SOLO los caracteres */del final
Guarda el archivo

🔢 REGLAS IMPORTANTES PARA ID
✅Rangos recomendados:
Fútbol próximo: 1-100
Vóley próximo: 101-200
Fútbol jugado: 201-300
Voleibol jugado: 301-400
❌ NO repetir identificaciones:
Javascript
Copiar
// ❌ MAL - ID duplicado 
{ id : 1 , deporte : "futbol" , ... },
{ id : 1 , deporte : "voley" , ... },    // ← ¡Error!

// ✅ BIEN - IDs únicos 
{ id : 1 , deporte : "futbol" , ... },
{ id : 101 , deporte : "voley" , ... }, // ← ¡Correcto!
📅 FORMATO DE FECHAS Y HORAS

Javascript
Copiar

Javascript
Copiar

✅ Horas correctas:
Javascript
Copiar

❌ Horas incorrectas:
Javascript
Copiar

⚠️ ERRORES COMUNES Y CÓMO EVITARLOS
❌ Error 1: Falta la coma
Javascript
Copiar
// MAL
{
  id : 1 ,
   deporte : "futbol" 
}   // ← Falta coma aquí
{
  id : 2 ,
   deporte : "voley"
}

// BIEN
{
  id : 1 ,
   deporte : "futbol" 
}, // ← Coma agregada
{
  id : 2 ,
   deporte : "voley"
}
❌ Error 2: Ganador no coincide con equipos
Javascript
Copiar
// MAL
{
  equipos : [ "Santa Rosa FC" , "Los Tigres" ],
   ganador : "Equipo Incorrecto"   // ← No está en la lista
}

// BIEN
{
  equipos : [ "Santa Rosa FC" , "Los Tigres" ],
   ganador : "Santa Rosa FC"       // ← Coincide exactamente
}
❌ Error 3: Nombres inconsistentes en goles
Javascript
Copiar
// MAL
{
  equipos : [ "Santa Rosa FC" , "Los Tigres" ],
   goles : {
     "Santa Rosa" : [...]   // ← Nombre incompleto
  }
}

// BIEN
{
  equipos : [ "Santa Rosa FC" , "Los Tigres" ],
   goles : {
     "Santa Rosa FC" : [...] // ← Nombre exacto
  }
}

RESUMEN RÁPIDO
Partidos próximos → Agregar directamente al array
Partidos jugados → Agregar dentro de /* */y luego descomentar
IDs únicos → No repetir números
Fechas → FormatoYYYY-MM-DD
Horas → FormatoHH:MM
Ganador → Debe coincidir exactamente con el nombre del equipo
